Connection Guide

Connection Diagram

Hardware Components

  • ESP32 Development Board
  • JSN-SR04T Ultrasonic Sensor
  • MAX7219 LED Matrix Display
  • Buzzer (optional, for alerts)
  • LED (optional, for alerts)

Pin Connections

Component Pin ESP32 Pin Color Code
JSN-SR04T Trigger TRIG GPIO 5 Red
JSN-SR04T Echo ECHO GPIO 18 Green
MAX7219 DIN DIN GPIO 23 Orange
MAX7219 CLK CLK GPIO 19 Purple
MAX7219 CS CS GPIO 21 Pink
Buzzer + GPIO 22 Teal
Alert LED + GPIO 2 Brown

Power Connections

  • JSN-SR04T VCC → 5V
  • JSN-SR04T GND → GND
  • MAX7219 VCC → 5V
  • MAX7219 GND → GND
  • Buzzer GND → GND
  • Alert LED GND → GND (with 220Ω resistor)

Installation Tips

  • Mount the ultrasonic sensor at the top of the tank, facing down
  • Ensure the sensor is perpendicular to the water surface
  • Keep the sensor away from tank walls to avoid false readings
  • Use shielded cables for long wire runs to reduce interference
  • Mount the display in a visible location
